# Solarium

Fresh walking skeleton: 3D solar system + classical aspect theater.

**Live:** https://shamanramen.github.io/solarium/

## Scope (this slice)

- Sun + 8 planets, Earth as geocentric observer  
- Real geocentric ecliptic longitudes (`astronomy-engine`)  
- Readable scale (labeled)  
- Major aspects: conjunction, sextile, square, trine, opposition  
- No natal form, no VR, no Asteria monorepo, no auth  

## Stack

Vite · TypeScript · Three.js · astronomy-engine · static deploy

## Commands

```bash
npm install
npm run dev
npm test
npm run build
```

## Frame

Always on-screen: **geocentric · aspects · readable scale**
